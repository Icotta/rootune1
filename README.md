# 🐝 Rootine — твой ритм

Планировщик жизни по фазам менструального цикла. Спорт, питание, работа — в согласии с твоим телом.

## ✨ Особенности

- 🗓️ **Календарь цикла** — визуальное отображение всех фаз
- 💪 **Трекер ресурсов** — физическая, когнитивная и эмоциональная энергия
- 📋 **Умные задачи** — задачи подстраиваются под твой цикл
- 🌱 **Персональные рекомендации** — что делать в каждой фазе
- 📱 **PWA** — работает как нативное приложение

## 🚀 Технологии

- Vanilla JavaScript
- LocalStorage для хранения данных
- PWA (Service Worker, Manifest)
- Адаптивный дизайн
